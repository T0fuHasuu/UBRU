**Exam Questions and Key Concepts**

### **Network Types and Components**

#### **Multiple Choice Questions**

1. What is the main difference between a Client-Server network and a Peer-to-Peer network?
   * A) Client-Server networks have a central server, while Peer-to-Peer networks do not.
   * B) Peer-to-Peer networks are more secure than Client-Server networks.
   * C) Client-Server networks are more complex to set up than Peer-to-Peer networks.
   * D) Peer-to-Peer networks are more scalable than Client-Server networks.

2. Which of the following is a benefit of a Peer-to-Peer network?
   * A) Easy setup and low complexity
   * B) Centralized management and control
   * C) High security and data protection
   * D) Ability to handle large-scale applications

3. What is the primary role of a Network Switch in a network infrastructure?
   * A) To connect devices and route network traffic
   * B) To provide wireless connectivity to devices
   * C) To store and manage network data
   * D) To enforce network security policies

#### **Short Answer Questions**

1. Explain the difference between a Client and a Server in a network.
2. Describe the purpose of a File Server in a network.
3. How does a Printer Server function within a network?

#### **Long-Form Questions**

1. Compare and contrast the advantages and disadvantages of Client-Server networks and Peer-to-Peer networks.
2. Discuss the role of a Communication Server in a network and its primary functions.
3. Explain the concept of multiple roles that a single device can play in a network.

### **Network Connectivity and ISP Services**

#### **Multiple Choice Questions**

1. What is the primary purpose of an Internet Service Provider (ISP)?
   * A) To provide network equipment and infrastructure
   * B) To connect local networks to the global internet
   * C) To offer technical support and troubleshooting
   * D) All of the above

2. Which of the following is a common additional service offered by ISPs?
   * A) Email accounts
   * B) Web hosting
   * C) Data backup solutions
   * D) All of the above

3. What is the main advantage of connecting to the internet through a router rather than directly with a modem?
   * A) Increased internet speed
   * B) Better network security
   * C) Easier configuration
   * D) All of the above

#### **Short Answer Questions**

1. Describe the role of an ISP in providing internet connectivity.
2. What are the key components of an ISP network infrastructure?
3. Explain the difference between a direct modem connection and a router-based connection to an ISP.

#### **Long-Form Questions**

1. Discuss the various types of internet connection options available to users, including their advantages and disadvantages.
2. Analyze the importance of selecting the appropriate internet connection method based on factors such as location, budget, and usage requirements.
3. Describe the process of connecting a home or small office network to the internet through an ISP, including the necessary hardware and configuration steps.
